# City Events — веб-приложение городских мероприятий (Django)

Учебный проект: веб-система для учета и отображения городских мероприятий.  
Пользователь может просматривать список событий и подробности о мероприятии, добавлять события в избранное, оставлять отзывы и оценки.  
Администратор управляет контентом через Django Admin.

---

## Функционал

- Просмотр списка мероприятий и страницы события
- Категории мероприятий
- Регистрация и авторизация пользователей
- Добавление/удаление событий в избранное
- Отзывы и оценки мероприятий (ограничение: 1 отзыв пользователя на 1 событие)
- Администрирование через Django Admin (добавление и редактирование мероприятий, категорий и т.д.)

---

## Технологии

- Python
- Django
- PostgreSQL (или SQLite — зависит от настроек проекта)
- HTML/CSS (шаблоны Django)

## Установка и запуск (Windows / PowerShell)

### 1) Клонирование проекта
```powershell
git clone <https://github.com/CYR3X77/city_events.git>
cd city_events_project

2) Создание и активация виртуального окружения
python -m venv venv
venv\Scripts\Activate.ps1

3) Установка зависимостей
pip install -r requirements.txt

4) Настройка базы данных
Перед запуском необходимо создать базу данных PostgreSQL и указать параметры подключения в settings.py.

Пример параметров подключения (ориентировочно):
DB_NAME: city_events_db
DB_USER: postgres
DB_PASSWORD: <ваш пароль>
DB_HOST: localhost
DB_PORT: 5432

Далее примените миграции:

python manage.py migrate

5) Создание администратора
python manage.py createsuperuser

6) Запуск сервера
python manage.py runserver

Доступ к системе
Главная страница сайта: http://127.0.0.1:8000/
Админ-панель: http://127.0.0.1:8000/admin/
